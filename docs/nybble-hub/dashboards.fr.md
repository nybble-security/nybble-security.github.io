# Tableaux de bord
Nybble fournit des tableaux de bord simples mais efficaces pour vous donner un aperçu de l'état de votre sécurité.

## Tableau de bord, tri par état
![Tableaux de bord](../img/nybble-hub/dashboards-1.png){: style="width:900px;display:block;margin:auto"}

Nybble affiche 4 métriques, déclinées par niveau de sécurité :

- Alertes reçues : toutes les alertes envoyées par le SIEM, quel que soit le statut. Le niveau de sécurité vous aide à voir la criticité des alertes émises par votre SI.
- En cours : Nybble travaille actuellement sur ces alertes ou attend les commentaires des clients.
- Investiguées : alertes clôturées, couvertes par une investigation.
- Faux positif : fermeture des alertes en les rejetant. Nybble s'efforce constamment de réduire ce nombre.
